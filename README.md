# Shaco

A League of Legends wrapper for the LCU REST & WS + the ingame API

| :exclamation:  Only tested on Windows |
|-----------------------------------------|

# How to install
Add the following snippet to your `Cargo.toml`
```toml
shaco = "0.6.0"
```
